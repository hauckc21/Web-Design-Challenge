Weather Data

The purpose of this study was to examine how proximity to the Equator impacts temperature. To answer this question, we performed a series of API calls to pull weather data from OpenWeatherMap.org for over 500 cities around the world. The data collected focused on max temperature, humidity, cloudiness, and wind speed for each city and also collected each city’s geo-coordinates. We then analyzed the data in Pandas and graphed each city’s weather against their respective latitudes using scatter plots in Matplotlib.
